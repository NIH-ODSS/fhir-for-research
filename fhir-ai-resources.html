Cumulus smarthealthit.org/cumulus SMART on FHIR, FHIR Bulk Access, Information Extraction, Data sharing, Data analysis, Federated networks Turnkey functions to annotate FHIR data for analytics, de-identify data, and query for cohorts. Uses FHIR APIs and NLP/LLMs to process population data for care, research, and public health. SMART on FHIR (Boston Children’s Hospital) The SMART Health IT project develops open technologies for digital health.
FHIR-GPT ai.nejm.org/doi/10.1056/AIcs2300301 FHIR, Few-Shot Learning Converts clinical narratives into FHIR resources using large language models. Developed FHIR-GPT for transforming clinical texts into FHIR medication statements. Northwestern University, Siemens Healthineers Peer-reviewed research from Northwestern and Siemens Healthineers.
Vertex AI Agent Builder cloud.google.com/generative-ai-app-builder Development tool Build AI agents and applications using natural language or a code-first approach. Alphabet Services for production tools using Google Cloud Platform.

RAG on FHIR github.com/samschifman/RAG_on_FHIR FHIR, Retrieval Augmented Generation Uses Retrieval-Augmented Generation (RAG) to combine FHIR with Generative AI. Independently developed Independent project.
LLM on FHIR github.com/StanfordBDHG/LLMonFHIR FHIR, Digital Health, Mobile Apps, LLMs Interprets, summarizes, and explains health record data using OpenAI GPT and FHIR. Stanford Biodesign Digital Health Group Develops digital health solutions and modular, interoperable tools.
HealthGPT github.com/StanfordBDHG/HealthGPT FHIR, Digital Health, Mobile Apps, LLMs iOS app that lets users interact with Apple Health data using natural language. Stanford Biodesign Digital Health Group Develops digital health solutions and modular, interoperable tools.
HealthChain github.com/dotimplement/HealthChain EHR Integration, NLP, Testing and Evaluation, Pipeline Open-source framework for scalable AI and NLP apps in healthcare. dotimplement Builds dev tools and consults on healthcare AI.
Doctor Dok github.com/CatchTheTornado/doctor-dok Storage, Digitization, LLM Query and Summarization Secure open-source framework for storing and managing medical records via LLMs. Catch the Tornado Open-source framework for record management.
pyomop github.com/dermatologist/pyomop FHIR, OMOP, Query and Analysis Python library to query OMOP CDM databases and convert results to DataFrames. Uses Llama Index. Independently developed Independent project.
MEDPrompt github.com/329162516/medprompt Text Conversion Prompt tools including FHIR2Text, which converts FHIR to plain text. Independently developed Independent project.
IRIS FHIRfy github.com/jrpereirajr/iris-fhirfy FHIR Resource Generator Converts raw data into FHIR resources using InterSystems IRIS platform and LLMs. Independently developed from Intersystems template Independent project.
